== Description ==
WordPress плагин для генерации и автоматической публикации комментариев и отзывов на вашем сайте. Работает на основе ChatGPT.

Наш плагин на базе ИИ может генерировать комментарии, которые выглядят естественно и релевантно, что поможет увеличить доверие и лояльность посетителей. Автоматическая написание комментариев и отзывов  сэкономит время и улучшит эффективность вашей коммуникации, публикация происходит в автоматическом режиме. 

Плагин позволяет в автоматическом режиме добавлять новые комментарии на страницы сайта, таким образом делая сайт более живым. Комментарии и отзывы создаются на основе Title или H1, что делает их текст релевантным тексту страницы.

В настройках вы можете указать типы страниц на которых публиковать комментарии, а так же список конкретных URL. Возможно задать ограничение на количество публикаций в сутки. Так же генерировать комментарии и отзывы можно с различным тоном, например позитивным или негативным.

Плагин работает с использованием API, это позволяет отправлять запросы на генерацию контента и получать обратно сгенерированый текст для публикации. Обмен API запросами происходит через наш домен https://aiwppost.com.
